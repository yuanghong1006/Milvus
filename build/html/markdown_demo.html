儒家思想

  • 核心：仁、义、礼、智、信

  • 代表人物：

    • 孔子：”己所不欲，勿施于人”

    • 孟子：”民为贵，社稷次之，君为轻”

“大学之道，在明明德，在亲民，在止于至善。” ——《大学》

道家思想

  • 核心概念：

    • 道法自然

    • 无为而治

  • 经典著作：

    • 《道德经》

    • 《庄子》
